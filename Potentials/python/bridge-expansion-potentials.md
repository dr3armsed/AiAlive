# Python Bridge Expansion Potentials

- Sensory shaping before text fallback for emotion selection.
- Theory/oracle/artifact hint generation as composable side channels.

## Why this leads to something else

- These are currently deterministic hinting channels.
- They can evolve into model routing policy, safety throttles, and context budgeting logic.

## Promotion criteria

- Add explicit contract versioning for bridge response schema.
- Add safety policy checks before tool-oracle hint promotion.
